# Project-Roster

The objective of the project is to create an application/program that automatically prepares an employee shift roster based on the preferences. This roster should also adhere mandatory conditions such as time-offs.



Conditions Identified so far:
1.	Each employee should only work 5 days in a week.
2.	Priority 1 employees should be on fixed shifts all time
3.	Priority 2 employees should have fixed shifts as priority but are available for other shifts if need be.
4.	Priority 3 employees can be allocated with all shifts.
5.	Night shifts should always be in consecutive days of more than 3 but not exceed 5
